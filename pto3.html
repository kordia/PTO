PTO v3.0

A simple PTO (Paid Time Off) calculator that uses how many years you worked to get correct PTO rate. Then uses how many weeks you have worked so far this year to calculate total PTO to date. Then deducts any PTO that you have taken to date. Results can be returned in hours or days or weeks.

  1. START
  2. SET "how many years worked"
  3. Diplay number of years
  4. SET "how many weeks you have worked"
  5. Display number of years worked
  6. SET "how many years worked"
  7. Display Deducted number of years
  8. SET number of years deducted
  9. Display any deducted PTO to date
  10. END
